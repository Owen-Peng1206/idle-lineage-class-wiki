# 吉爾塔斯流程攻略

## 攻略

1. 去拉斯塔巴德-格蘭肯神殿.長老室 打 "死亡騎士之書"（不是死亡騎士之印記）
2. 去拉斯塔巴德-長老會議廳 找真.冥皇丹特斯，選受詛咒的黑暗妖精聖殿（消耗死亡騎士之書×1，進去一次扣一次），裡面boss就是吉爾塔斯
3. 拉斯塔巴德-長老會議廳找 亞提歐斯製作完整的召喚球是用來保留吉爾塔斯當前血量，離開時扣一（作者有說明過）
4. 完整的召喚球怎麼做？
   答：去拉斯塔巴德-長老會議廳 找真.冥皇丹特斯，黑暗妖精聖殿（消耗死亡騎士之書×1，進去一次扣一次），裡面有召喚球之核跟召喚球碎片
   更正，拉斯塔巴德-魔獸訓練場 地獄奴隸也會掉落召喚球之核跟召喚球碎片（應該是作者漏了它😅）
5. 另外，打死吉爾塔斯會掉吉爾塔斯的封印，去拉斯塔巴德-長老會議廳 找真.冥皇丹特斯交出吉爾塔斯的封印，會進去打真.死亡騎士 真冥皇丹斯特（消耗吉爾塔斯的封印×1，進去一次扣一次）

## 程式碼驗證結果

經過比對遊戲主程式資料庫，該攻略 **完全屬實，且非常精確**。以下是針對每一點的程式碼驗證細節：

### 1. 打「死亡騎士之書」：✅ 屬實
* **資料庫驗證：** 根據 `01-drops-config.js` 的掉落設定，在「格蘭肯神殿．長老之室」中的 3 種一般怪與 8 隻長老 BOSS，確實全都有 `1%` 的機率會掉落「死亡騎士之書」（`item_dk_book`）。

### 2. 找真.冥皇丹特斯進受詛咒聖殿打吉爾塔斯：✅ 屬實
* **資料庫驗證：** 根據 `00-data.js` 中對「真．冥皇丹特斯」(`npc_dantes_lord`) 的設定，對話並交出「死亡騎士之書」即可選擇進入「黑暗妖精聖地」或「受詛咒的黑暗妖精聖地」。
* **BOSS設定：** 選擇進入「受詛咒的黑暗妖精聖地」後，裡面的純 BOSS 房確實就是對戰「吉爾塔斯」(`sanct_giltas`)。每次進入固定消耗 1 本。

### 3. 製作「完整的召喚球」保留血量：✅ 屬實 (僅有微小譯名差異)
* **資料庫驗證：** NPC 確實在長老會議廳，程式內的正式譯名為 **「亞提利歐」**（網友記成亞提歐斯，為同一個 NPC）。
* **功能設定：** 根據 `00-data.js` 描述與 `03-combat-core.js` 的邏輯，帶著「完整的召喚球」挑戰吉爾塔斯，若以任何方式離開（回村、瞬移、切圖、戰敗），都會消耗 1 顆召喚球，但會 **暫停吉爾塔斯的回血，將剩餘 HP 封印在原地**。若身上沒有球而離開，重新進入將面對滿血重置的全新吉爾塔斯。

### 4. 完整的召喚球材料及掉落處：✅ 屬實 (網友更正的點完全正確)
* **資料庫驗證：** 合成需要「召喚球之核」與 4 個「召喚球碎片」。
* **掉落來源驗證：** 在 `01-drops-config.js` 的掉落清單中，「受詛咒的黑暗妖精聖地」裡的小怪（如受詛咒的黑暗妖精鬥士/法師/騎士、食腐獸、特提斯、翼龍等）確實都有這兩樣材料的掉落機率。
* **地獄奴隸驗證：** 網友提到的更正內容完美命中。程式內 `地獄奴隸` 的掉落列表確實包含了 `['mat_summonorb_core',0.1]` (召喚球之核 0.1%) 與 `['mat_summonorb_shard',1]` (召喚球碎片 1%)。

### 5. 隱藏流程：打死吉爾塔斯後挑戰「真冥皇丹特斯」：✅ 屬實
* **資料庫驗證：** `01-drops-config.js` 設定吉爾塔斯死亡時，會以 **100% 的機率**掉落「吉爾塔斯的封印」(`item_giltas_seal`)。
* **後續驗證：** `00-data.js` 設定，將此封印交給長老會議廳的「真．冥皇丹特斯」，會消耗 1 個封印並被傳送至「崩壞的長老會議廳」。這個純 BOSS 房的頭目就是 **真‧死亡騎士 冥皇丹特斯** (`sanct_dantes`)，每次挑戰同樣會消耗 1 個。

# 組隊攻略建議
根據遊戲代碼中針對「吉爾塔斯」( sanct_giltas ) 的素質與技能設定，這是一場極高難度、高度考驗團隊抗性與應變能力的戰鬥。以下是為您量身打造的組隊攻略建議： 
                                                                                                                                                          
  ### 📊 吉爾塔斯 核心素質解 析                                                                                                                            
                                                                                                                                                          
  • 等級/體型/種族： Lv.99 / 大型 (L) / 異界生物                                                                                                          
  • 血量 (HP)： 440,000 (非常厚實，需長期戰鬥)                                                                                                            
  • 防禦面： AC -99 (極高物防)，MR 300 (幾乎魔法免疫)                                                                                                     
  • 攻擊面： 基礎傷害 5~158，額外傷害(DB) 403，命中 132。普攻傷害極其恐怖，且帶有  noDmgCurve: true  (無視部分減傷公式)，拳拳到肉。                       
  • 狂暴機制： 當 HP 低於 30% (132,000) 時，命中與傷害會乘上 1.2倍，進入極度危險的斬殺期。                                                                
  ──────                                                                                                                                                  
  ### ⚔️ 致命技能與應對策略                                                                                                                               
                                                                                                                                                          
  吉爾塔斯擁有四招毀滅性的技能，且部分為全體攻擊 (AoE)：                                                                                                  
                                                                                                                                                          
  1. 岩漿流星雨 (全體火屬性必中攻擊 + 灼燒)                                                                                                               
      • 機制： 傷害極高 (DB 500) 且設定為  alwaysHit: true  (必中)。並附帶長達 18 秒、每 3 秒扣 200 血的「灼燒」持續傷害。                                
      • 應對： 全隊必須撐高 火屬性抗性。補師需要預判此技能的施放節奏，施放全體治癒術或強力補血，血量較少的遠程職業隨時注意血線。                          
  2. 毒氣風暴 (全體中毒)                                                                                                                                  
      • 機制： 長達 18 秒的範圍中毒，每 3 秒扣 200 血。                                                                                                   
      • 應對： 中毒配合灼燒會造成巨量每秒傷害 (DoT)，必須攜帶解毒藥水，或依靠法師/妖精的解毒技能迅速解除。                                                
  3. 沙塵暴 (範圍沉默)                                                                                                                                    
      • 機制： 使範圍內的目標無法施展魔法。                                                                                                               
      • 應對： 如果法師或水妖精被沉默，團隊會瞬間斷補崩盤。後排務必撐高 MR (魔法防禦) 來抵抗異常狀態，並備妥能解除沉默的消耗品。                          
  4. ⚠️ 血壁空間 (最危險的團滅機制)                                                                                                                       
      • 機制： 吉爾塔斯給自己上 Buff（持續 10 秒），隨機獲得 「近戰反射」、「遠程反射」或「魔法反射」 其中一種。                                          
      • 應對： 反彈的傷害與玩家打出的傷害完全相等，且「無視絕對屏障」！                                                                                   
      一旦吉爾塔斯開啟血壁空間，對應輸出類型的玩家 必須立刻停手 (暫停攻擊)，否則後期玩家的高傷害打在自己薄弱的血量上，絕對是瞬間被自己秒殺。

  ### 🛡️ 屬性與防禦要求                                                                                                                                   
                                                                                                                                                          
  • 火屬性抗性： ★★★★★ (應付岩漿流星雨)                                                                                                                   
  • 魔法防禦 (MR)： ★★★★★ (對抗沉默與毒氣，盡量大於 150 以上甚至更高)                                                                                     
  • 近戰命中： ★★★★☆ (對抗 AC -99，命中不夠會一直揮空)

  ### 👥 隊伍組成建議 (全自動戰鬥特化配置：1主角 + 3傭兵 + 寵物 + 召喚獸)
  
  由於本遊戲為全自動戰鬥，玩家 **「無法手動停手」**，面對吉爾塔斯高達 300 的 MR 與必定會反傷死人的「血壁空間」，隊伍配置必須以「分散滅團風險」與「提高生存容錯」為核心：
  
  1. 傷害類型極度分散化 (雞蛋絕不能放在同個籃子)：
      • 絕對避免純近戰或純遠程隊伍！吉爾塔斯的「血壁空間」會隨機反射一種傷害（近/遠/魔）。如果全隊都是物理近戰，一旦碰到近戰反射，全隊會在一秒內把自己秒殺。
      • 建議配置：1 坦 (近戰) + 1 弓妖 (遠程) + 1 法師/水妖 (魔法) + 主角 (視傭兵補齊缺少的傷害類型或擔任副補)。這樣每次血壁空間發動，最多只會反彈死一至兩名隊員，保留殘存戰力。
  2. 輔助與復活是核心 (法師/水妖精)：
      • 法師 或 水妖精 傭兵是這場戰鬥的 MVP。由於法師平A傷害極低，就算撞到「魔法反射」也不太會死。
      • 他們負責扛起「岩漿流星雨」與「毒氣風暴」的超高 AoE 傷害，自動群補與解毒，並在隊友被血壁反彈死時，將他們重新復活。
  3. 寵物與召喚獸 (不佔資源的消耗品)：
      • 寵物與召喚獸在此關卡極其脆弱，但在全自動戰鬥中，他們能稍微分散 BOSS 開場的火力，或提供短暫的常駐 Buff。死亡是常態，不需為此亂了陣腳。
  
  ---
  ### 💡 戰鬥流程與注意事項 (善用「回村」機制)
  
  既然無法微操停手，「回村 (撤退)」與「完整的召喚球」就是攻略吉爾塔斯唯一且最強的武器！
  
  1. 開場整備： 挑戰前確保包包內擁有複數以上的 **「完整的召喚球」**，這是全自動戰鬥下「磨死」BOSS 的關鍵。
  2. 常規階段 (100% ~ 30%) 的血壁應對：
      • 戰鬥全自動進行，當吉爾塔斯開啟「血壁空間」時，對應的輸出隊友極高機率會因為無法停手而反彈陣亡。
      • 此時仰賴法師/水妖傭兵自動進行復活。若復活不及、補血跟不上，或者主坦不幸倒地導致 BOSS 轉頭撲向後排，請**立刻手動按下「回村」**！
  3. 召喚球「接關」戰術：
      • 只要您持有「完整的召喚球」，回村時只會消耗 1 顆，但 **吉爾塔斯的殘血狀態會被完美保留，且停止回血**。
      • 回村後，將全隊重新復活、補滿血量、重新上好 Buff 與火屬性防護，接著再次進入聖地。您會發現剛才被打掉的血量並沒有恢復，利用這種「無限接關」的添油戰術，即便是全自動戰鬥也能強行將 BOSS 磨進狂暴期。
  4. 狂暴階段 (低於 30%) 的斬殺：
      • 吉爾塔斯傷害與命中再提升 20%，主坦幾乎扛不住。這個階段就是單純的傷害競速，如果再次崩盤，就果斷再次回村消耗召喚球，重複「滿狀態進場打一波爆發 -> 滅團前逃跑」的循環，直到將其擊殺！
